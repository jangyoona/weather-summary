# weather-summary
<br/>

☀️ 간단한 날씨 요약 정보를 조회하는 Vue 프로젝트입니다. ☀️

<br/>
<br/>

## 🚀 run

<br/>

### 1. 패키지 설치
```
npm install
```

<br/>
<br/>

### 2. 환경 변수 설정
`env/.env` 파일을 생성한 후 아래 내용을 추가합니다.
```
VITE_API_URL=http://localhost:포트
```
<br/>
<br/>

### 3. 실행
```
npm run dev
```

<br/>
<br/>

### 4. 빌드
```
npm run build
```
